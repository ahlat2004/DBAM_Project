# Semester Complex Computing Problem / Project Requirements
## Course: Database Administration and Management (DAM) — Spring 2026

This document outlines the comprehensive requirements, deliverables, timelines, and grading criteria for the Complex Computing Problem (CCP) course project in Database Administration and Management (CLO-3).

---

## 1. Project Overview & Objectives
The goal of this course project is to experience database administration research and system development. It requires analyzing the latest database administration and management tasks to create a creative prototype along with formal documentation.

### Core Constraints
* **Focus:** The main focus must be strictly on **Database Administration and Management**.
* **Scientific Approach:** The project must contain a "science" component—there must be a general research question you are attempting to answer (e.g., *"Can distributed service X be efficiently implemented on system Y?"*).
* **Implementation & Evaluation:** Every project must include both substantial practical implementation and testing/experimentation parts to verify the solution.
* **Group Work:** Projects must be done in **groups of two** (individual projects are prohibited except in special cases subject to prior discussion).

---

## 2. Project Deliverables & Structure
The final deliverable must be submitted as a **Zip archive on the LMS** containing:
1.  **Final Project Report (PDF)**
2.  **Presentation Slides (PPT/PPTX)**
3.  **Source Code & Database Files (e.g., `.mdf` file)**
4.  **GitHub Link & LinkedIn Video Link** (must be embedded within the report)

### Report Guidelines
* **Typical Sections:** Abstract, Introduction, Body Sections, Evaluation/Testing, Conclusions, and References.
* **Architecture Diagram:** A graphical representation of the tool/system architecture is mandatory.
* **Code Explanation:** Do not perform large unannotated code dumps. Every SQL code listing included in the report must be accompanied by an explanation.
* **Contribution:** The effort of both group members must be clearly reflected in the report.

### Extra Mandatory Tasks
* **GitHub Repository:** Upload project files and include the link in the final report.
* **LinkedIn Video:** Upload a short video (1–2 minutes) demonstrating or describing the project along with a description on LinkedIn.
* **Project Gala:** Mandatory physical participation in the Project Gala event (Date, time, and venue to be announced).
